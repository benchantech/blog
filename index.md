<section class="intro narrow">
  <h1>BenChanTech is no longer a portfolio website. It is the map of a living reasoning ecosystem.</h1>
  <p>Its purpose is not to explain every project individually. Its purpose is to show how each property contributes a different function without forcing every property to satisfy the same standard of evidence.</p>
  <p>Some work preserves lived experience. Some discovers recurring structures. Some distills durable doctrine. Some operationalizes that doctrine. Some exists simply to grow without governance.</p>
  <p>This is where those relationships stay visible.</p>
</section>
