# BenChanTech

Next.js source for [benchantech.com](https://benchantech.com), the routing foyer for the Ben Chan Tech LLC ecosystem.

## Stack

- Next.js App Router
- React
- TypeScript
- Vercel

## Development

```bash
npm install
npm run dev
npm run build
npm test
```

Keep content in the Next app, typed content modules, and public assets under `public/`.
